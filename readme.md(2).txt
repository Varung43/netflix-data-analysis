📺 Netflix Data Analysis Project

This project analyzes the Netflix content library using Python libraries like Pandas, Seaborn, and Matplotlib. It explores trends by content type, genres, countries, and year-wise growth.

📊 Key Analysis:
- Total count of Movies vs TV Shows
- Content added each year
- Top 10 content-producing countries
- Most common genres
- Top directors on Netflix

🧰 Tools Used:
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

🔍 Dataset:
- Source: Kaggle [Netflix Shows Dataset](https://www.kaggle.com/datasets/shivamb/netflix-shows)
- File: `netflix_titles.csv`

💡 Insights:
- Movies outnumber TV Shows on Netflix
- Majority of content is from the US and India
- Netflix added the most content around 2019–2020
- Drama and International Movies are the most common genres
